Capabilities

Everything you need, nowhere near your way.

Notch UI
Animated overlay that expands from the MacBook notch. Stays out of your way until it matters.
Live Session Monitoring
Track multiple Claude Code and Codex sessions simultaneously. See what every agent is doing right now.
Permission Approvals
Approve or deny tool executions directly from the notch. No terminal switch needed.
Chat History
Full conversation history with markdown rendering. Review the full context without leaving your flow.
Auto-Setup
Hooks install automatically on first launch. Zero configuration. Open the app and start working.
macOS Native
Built for macOS 15.6+. Uses native notch space. No menu bar clutter, no separate window.
Process

02

Open the app. Hooks are installed automatically into ~/.claude/hooks/. Start a Claude Code or Codex session.
03
Approve
